Use Case

When battery SoC drops below 20%, the EV Range Extender activates POWER SAVE mode:

  • Ambient lights dimmed to 30%
  • Seat heating disabled
  • Range computed at degraded efficiency

Above 20% SoC, all systems return to NORMAL.
